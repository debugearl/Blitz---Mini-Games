BLITZ | A Mini Games Web Collection by Arcadia

Welcome to BLITZ: A Mini Games Web Collection!
This project provides online games accessible through the web, which hosts multiple fun and interactive mini games. The project allows users to browse and choose the game they want to play, which direct them to the game immediately. No downloads, no installation, and no complicated setups required to do.

About the Project
The website was created in fulfillment of the final project requirement for the course IPT 313: Integrated Programming Technologies 1. It was also created as practice and a  demonstration of web development, game logic, and user interaction in the browser. Additionally, it serves as a portfolio piece to show coding and design skills.

Features
• Multiple mini games in one website
• Playable on smartphones, tablets, and desktops
• No installation required
• Games are simple, fast to load, and easy to understand
• Inspired by widely known and popular game concepts

Included Games
• Flap Tap
• Whack a Mole
• Duck Hunt
• Meme Matching Game
• Stack Block
• Love Calculator

How to Use
1. Download the folder containing all the files.
2. Open index.html in any browser.
3. Choose a game from the arcade and start playing!

Online Deployment
This project was also deployed online for easy access. However, since it is hosted on a free platform, some games may experience loading delays, lags, or reduced performance. For best experience, we recommend downloading and running the project locally. 
Live Version: https://blitz-mini-games.netlify.app/
